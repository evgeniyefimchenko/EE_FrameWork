# Changelog

Все заметные изменения публичных версий EE_FrameWork фиксируются в этом файле.

Формат ориентирован на человекочитаемый релизный журнал для русскоязычных пользователей.

## v1.0.0 - 2026-03-28

Первый публичный релиз EE_FrameWork как рабочего продукта.

### Что вошло в релиз

- MVC-ядро с front controller, Router, View и Layout-контуром.
- Встроенная административная панель для управления пользователями, категориями, страницами, свойствами и наборами свойств.
- Контентная модель `property types -> properties -> sets -> category types -> categories -> pages`.
- Мультиязычная модель сущностей с отдельным translation-layer для страниц и категорий.
- Публичные semantic URL для категорий и страниц.
- WordPress-импорт с поддержкой curated mapping, сложных свойств и фоновой очереди media mirror.
- Cron-агенты, health-screen и operational alerts.
- Очередной backup-контур с планами резервного копирования и удалёнными целями `SFTP/FTP`.
- Legal-контур для политики обработки персональных данных и согласия пользователя.
- Шаблонный `configuration.php` для чистого clone/install-сценария.
- Release-ready очистка репозитория от runtime-данных и project-specific миграционных артефактов.

### Важные релизные принципы

- В репозиторий не входят `cache/`, `logs/`, `uploads/` и другие runtime-данные.
- `inc/configuration.php` хранится в репозитории как шаблон, а не как боевой конфиг.
- Канонический домен проекта задаётся в `configuration.php`, а не зашивается в `.htaccess`.
